JADE BENEFITS  -  WHERE TO PUT YOUR PHOTOS
==========================================

Drop your photos into this folder (assets/images/) using the exact
file names below. As soon as a file with the matching name exists,
it will appear on the site automatically. Until then, a clean green
placeholder block shows in its place, so the site never looks broken.

Recommended file names (JPG or PNG, landscape unless noted):

  hero.jpg              Home hero image  (people / business owners, ~1200x900)
  group-coverage.jpg    Group coverage card  (~800x600)
  truckers.jpg          Truckers card / blog hero  (~800x600)
  selfemployed.jpg      Self-employed / 1099 card  (~800x600)
  families.jpg          Families card  (~800x600)
  smallbusiness.jpg     Small business owners card on Home  (~800x600)
  trades.jpg            Trade industries  (~800x600)
  medical.jpg           Medical groups  (~800x600)
  ppo.jpg               PPO plans card on Services  (~800x600)
  professional.jpg      Professional services card on Services  (~800x600)
  enrollment.jpg        Dedicated enrollment card on Services  (~800x600)
  about.jpg             About page image  (~1000x800)

TIP: keep files under ~400 KB each for fast loading. You can compress
them free at squoosh.app or tinypng.com.

To swap a name, open the HTML file, find the <img src="assets/images/...">
line, and change the file name to match yours.
